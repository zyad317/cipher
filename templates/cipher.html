Quick Reference
{% if cipher_name == 'caesar' %} Each letter is shifted by the key value. A with key=3 becomes D. The key can be 1–25. Non-alphabetic characters are preserved unchanged. {% elif cipher_name == 'autokey' %} The key is extended using the plaintext itself, making it harder to crack than Vigenère. Encode and decode require the same initial keyword. {% elif cipher_name == 'vigenere' %} A repeating keyword shifts each letter differently. 'KEY' repeats as KEYKEYKEY... against your plaintext for polyalphabetic substitution. {% elif cipher_name == 'railfence' %} Text is written diagonally across N rails then read row by row. More rails = more complex cipher pattern. {% elif cipher_name == 'monoalphabetic' %} Each letter maps to a fixed unique substitute. The alphabet mapping is fixed: a→o, b→z, c→m, etc. {% elif cipher_name == 'playfair' %} Letters are paired and encrypted using a 5×5 Polybius square built from your keyword. J is treated as I. {% elif cipher_name == 'hill' %} Uses matrix multiplication mod 26. Key length must form a perfect square (e.g. 4 chars → 2×2, 9 chars → 3×3). {% elif cipher_name == 'rowtransposition' %} Columns are rearranged according to the alphabetical order of key digits or letters. Key can be numeric or alphabetic. {% endif %}
